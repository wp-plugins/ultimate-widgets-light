Ultimate Widgets plugin allows you to add the most popular widgets like Contact Info, Facebook Page Plugin, Google Map, Testimonials, Twitter, etc...

== Description ==

Customize your sidebar with our awesome widgets, very easy to use, essential, unique & beautiful [PLUGIN DEMO](http://khositeweb.com/preview/ultimate-widgets/).

= Light Version =

* About Me Widget - A widget that talks to you.
* Contact Info Widget - Add contact info, phone, email, etc.
* Flickr Stream Widget - Images from your Flickr account.
* MailChimp Widget - Displays subscription form.
* Menu Widget - Displays a custom menu.

== Installation ==

1. Upload the entire `ultimate-widgets-light` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

All widgets are in Widgets in your WordPress admin panel.
